# Toy News



### This is an Android Application provides news from various countries and sources.

### It is made to be used as an example to build modern android application.

---

## Keywords
- MVVM architecture pattern
- DI with Hilt
- RxJava
- Retrofit
- Repository Pattern
- Navigation Component
- Single Activity
- Motion Layout 
- and Kotlin
